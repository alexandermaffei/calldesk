# **App Name**: CallDesk Lead Manager

## Core Features:

- Lead Dashboard: Displays a list of all leads with key information, filtering, and search options.
- Lead Detail View: Allows viewing and modification of individual lead details based on the Airtable fields.
- Airtable Integration: Connects to the Airtable base to fetch and update lead data.
- Call Log: Logs incoming calls, managed by the call center, to Airtable

## Style Guidelines:

- Primary color: Deep purple (#7209B7) for a modern and professional feel.
- Background color: Off-black (#071108) for a dark theme to minimize eye strain.
- Accent color: Soft yellow (#FFE787) for highlighting key elements and CTAs.
- Body: 'Inter' sans-serif font for a clean and modern feel.
- Headings: 'Space Grotesk' sans-serif font for impactful headlines.
- Use simple, clear icons to represent different lead statuses and actions.
- Clean and intuitive layout with clear information hierarchy.
